# 内容分享应用后端微服务

## 1. user-service

用户微服务 主要提供

- 用户登录
- 查询/修改用户信息
- 用户收藏、投稿、积分记录查询

## 2. content-service

内容微服务 主要提供

- 查询所有分享内容
- 查询指定的分享内容
- 新增投稿
- 审核投稿

## 3. notice-service

公共微服务 主要提供

- 管理员查询所有公告 显示/隐藏某个公告
- 新增、删除公告
- 查询最新公告
- 查询轮播广告

## 4. gateway-service

网关微服务 主要提供

- 路由转发
- 动态路由
- 统一认证鉴权
- 统一跨域处理
